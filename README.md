<hr>
<pre><code>你可以在自定义按键设置中取消或者重新设定默认快捷键，每一行输入以下设定命令:
</code></pre>
<ul>
<li><code>map key command</code>:将命令映射到vimium中，并且覆盖Chrome的默认设置</li>

<li><code>unmapAll</code>: 取消所有命令绑定，当你尝试删除vimium的所有命令并且开始自己的设定</li>
</ul>

</code></pre>
